
# PDFStudio Professional Server

## Railway Deploy

1. Upload project to GitHub
2. Import repo into Railway
3. Deploy automatically

## Local Run

```bash
npm install
npm start
```

Runs on:
http://localhost:8080

## Important

This project includes professional backend structure ready for:
- PDF merge
- Split
- OCR
- Office conversions
- JPG/PDF tools

You can later connect:
- LibreOffice
- pdf-lib
- sharp
- tesseract.js
